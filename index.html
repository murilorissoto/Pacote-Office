Pacote Office

Word Excel PowerPoint Outlook Acess OneNote

O Pacote Office é um conjunto de programas desenvolvido pela Microsoft para uso em escritórios e empresas. Ele inclui vários programas úteis que ajudam a criar, editar, formatar e compartilhar documentos, planilhas e apresentações. Vamos ver mais sobre cada um deles:

Microsoft Word: É um processador de texto que permite criar documentos com formatação, imagens, tabelas e gráficos. Ele é usado principalmente para criar cartas, relatórios, currículos e outros documentos.

Microsoft Excel: É um programa de planilha eletrônica que ajuda a organizar e analisar dados. Ele pode ser usado para fazer cálculos, gráficos e relatórios financeiros.

Atalhos:

DESCRIÇÃO COMANDOS
Selecionar intervalo até a última célula no canto inferior direito Ctrl+Shift+End
Alternar para célula anterior ou opção anterior na caixa de diálogo Shift+Tab
Mover uma tela para a direita Alt+PgDn
Abrir a guia Inserir e inserir tabelas dinâmicas, gráficos, suplementos, minigráficos, imagens, formas, cabeçalhos ou caixas de texto. Alt+N
Abrir a guia Exibir e Visualizar quebras de página e layouts, mostrar e ocultar linhas de grade e títulos, definir a ampliação do zoom, gerenciar janelas e painéis, e exibir macros. Alt+W

Microsoft PowerPoint: É um programa de apresentação que ajuda a criar slides com imagens, texto e gráficos para apresentações empresariais, acadêmicas e outros fins.

Atalhos:

DESCRIÇÃO COMANDOS
Criar nova apresentação. Ctrl+O
Recortar o texto, objeto ou slide selecionado. Ctrl+X
Alterar o tamanho da fonte para o texto selecionado. ALT + H, F, S
Adicionar um novo slide. Ctrl+M
Selecionar um layout de slide. Alt+H, L

Microsoft Outlook: É um programa de gerenciamento de e-mails que permite gerenciar suas mensagens, calendários e contatos. Ele também pode ser usado para agendar reuniões e enviar convites.

Atalhos:

DESCRIÇÃO COMANDOS
para responder com uma solicitação de reunião. Ctrl+Alt+R
para encaminhar uma mensagem. Ctrl+F
marcar uma mensagem como não sendo lixo eletrônico. Ctrl+Alt+J
exibir o conteúdo bloqueado em uma mensagem. Ctrl+Shift+I/td>
subir um arquivo em uma pasta. Ctrl+Shift+S

Microsoft Access: É um programa de gerenciamento de banco de dados que permite criar e gerenciar bancos de dados. Ele é usado principalmente para armazenar informações de clientes, fornecedores e outros dados importantes para uma empresa.

Atalhos:

DESCRIÇÃO COMANDOS
Exibir o menu de atalho do item selecionado. Shift+F10 ou tecla do Menu do Windows
Mova o foco para um painel diferente da janela. F6
Abra um banco de dados existente. Ctrl+O ou Ctrl+F12
Mostrar ou ocultar o Painel de Navegação. F11

Microsoft OneNote: É um programa de anotações que permite criar e organizar anotações em um formato de bloco de notas digital. Ele é usado principalmente para coletar ideias, notas e informações importantes.

Atalhos:

DESCRIÇÃO COMANDOS
Encaixar a OneNote janela.. Ctrl+Alt+D
Refazer a ação anterior, se possível. Ctrl+Y
Selecionar todos os itens na página atual. Ctrl+A Para expandir a seleção, pressione Ctrl+A novamente.
Desfazer a ação anterior. Ctrl+Z
O Pacote Office é uma ferramenta essencial para qualquer pessoa que trabalhe em um escritório ou empresa. Ele é fácil de usar e pode ser personalizado para atender às necessidades individuais de cada usuário. Espero que esta apresentação tenha sido útil para você!
